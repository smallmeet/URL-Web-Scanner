## Web URL Scanner

Web URL Scanner is a simple web app that uses WhatWeb to scan a given url and  shows specific details regarding the server such as::

- Http Headers
- Server Details
- Worpress Version, If Running Wordpress 
- Scripts & Libraries
- XSS Filters
- IP Address
- Cookies
- X Frame Options
